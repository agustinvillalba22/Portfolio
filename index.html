HIT COWORK

Web platform to manage coworking spaces: book rooms, register clients, handle contracts, add credits, and issue invoices in one streamlined system.

The challenge

The platform was cluttered, with too many paths to complete the same task, confusing users and making navigation less intuitive for beginners.

Solution

I redesigned the interface to simplify navigation and clearly define flows, reducing cognitive load for users. A consistent visual hierarchy highlights key actions, while contextual tooltips provide quick guidance. An AI-powered assistant supports inexperienced users step-by-step, from booking spaces to managing contracts, adding credits, and issuing invoices.
